# Dark Mode
Toggle the **macOS** dark mode from [**Spotlight Search**](https://support.apple.com/en-us/HT204014)

<br /><br /><br />
</div>

## Install

### Via [:beer: Homebrew Cask](//brew.sh)

```powershell
brew install --no-quarantine artginzburg/tap/darkmode
```
