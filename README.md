## Usage

Simply run the app.
I like to do it from [Spotlight](//support.apple.com/en-us/HT204014):

1. Press <kbd>âŒ˜</kbd> + <kbd>space</kbd> (or similar) to open Spotlight.
2. Start typing `dark`, continue until `Dark Mode` appear in the Top Hit section.
3. Hit <kbd>Enter</kbd>.
